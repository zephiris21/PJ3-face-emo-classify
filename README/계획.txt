감정과 관련해서 

- 눈물이 쏟아질 듯 슬픈 동시에 사람들에 대한 감사한 마음이 차오르는 복합적인 감정

이런 복합적인 감정을 인식하고, 그걸 언어로 표현할 수 있으면 좋겠다.



1단계는 현재 존재하는 감정 분석 모델인 FER2013, AFFECTNET, RAF-DB, CK+ 
의 데이터셋을 사용해서 만든 분석 모델을 활용해서 감정분류를 하는것

2단계는 한국인 얼굴을 추가 레이블링해서 학습 ( fine tunning ) 해서 한국인 감정인식 을 잘할 수 있는지 개선사항을 보는것

3단계는 한국인 얼굴을 추가 감정 분류를 조금 더 복합적으로 할 수 있도록 레이블 추가해서 인식할 수 있도록 전이학습을 해보려고 함.

4단계는 복합적인 감정을 레이블링 없이 분류하고 그걸 언어적으로 표현할 수 있는 모델을 만들 수 있을 지 알아보려고함.


우선 

- 자료조사

- 한국인 얼굴 데이터 수집
